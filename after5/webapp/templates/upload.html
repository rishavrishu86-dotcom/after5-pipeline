Import Leads

Drop a curated CSV from AI Ark (or any other source). The enrichment layer picks up new rows automatically on the next enrich.

Companies CSV

Header row required. Columns:

domain,name,country,icp[,source]

country must be UK. icp is free-form but the qualifier prefers the brief's labels: real_estate, mortgage_broker, car_dealer, solar_home_improvement, car_finance, recruitment_agency, gym_fitness.

People CSV (optional)

Use this when AI Ark / your source has named contacts attached. Companies must already exist.

domain,first_name,last_name,role[,email]

If email is blank we generate first.last@domain + MX-validate.

What happens next

  1. Rows go into companies table with status='new'
  2. Click Run Pipeline on the dashboard, or trigger enrich on Automation
  3. Each company gets scored on 6 signals via live HTTP — this is the Apify equivalent
  4. qualify buckets them into hot / warm / cold
  5. find-contacts + personalise set them up for the sequence
