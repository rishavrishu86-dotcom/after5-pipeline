Order of operations

  1. Buy {{ domain }} at Porkbun (~£12/yr for .co.uk).
  2. Set up Google Workspace on it (£4.60/mo Starter). Google's wizard tells you which MX records to add.
  3. Once Workspace is active: Admin Console → Apps → Google Workspace → Gmail → "Authenticate email" → generate the DKIM key. Paste the value into google._domainkey TXT record above.
  4. Verify with MXToolbox: SPF lookup, DKIM lookup, DMARC lookup — all should resolve.
  5. Warm up: send 5/day for a week, 20/day next week, 80/day after that. Skipping warmup = blacklist by week 2.
  6. Set SMTP_USER and SMTP_PASS in Render to the new Workspace account + a Gmail App Password.
  7. Run send-dry on the Automation page → check the rendered output looks right → flip to send-live.

DKIM is not generated by us — Google Workspace signs outgoing mail with its own key. We can only tell you where the record goes; the actual value comes from Google's Admin Console.
