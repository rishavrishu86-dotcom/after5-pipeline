Login password (best practice)

For production, use APP_PASSWORD_HASH instead of plaintext. Generate locally:

python -c "from werkzeug.security import generate_password_hash as h; print(h(input('password: ')))"

Paste the long output into Render → Environment → APP_PASSWORD_HASH. Once set, the plaintext APP_PASSWORD is ignored.
